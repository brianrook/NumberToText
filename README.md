# NumberToText converter

## Build and Run
run:

`mvn clean package`

`cd target`

`java -jar numberToText-0.0.1-SNAPSHOT.jar`

## Sample Execution
`Enter the number (type "end" to quit)`

 `1234.56` 

 `One thousand two hundred thirty-four and 56/100 dollars`

 `Enter the number (type "end" to quit)`

 `end`

 `done`
